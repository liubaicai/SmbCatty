# What is SmbCatty

SmbCatty is a modern, cross-platform SMB/CIFS client for browsing and managing Windows network shares. It provides an intuitive file browser interface for connecting to SMB servers and transferring files.

## Features

- **SMB/CIFS Protocol Support** - Connect to Windows shares, NAS devices, and Samba servers
- **Dual-Pane File Browser** - Local and remote file management with drag-and-drop
- **Host Management** - Save and organize your SMB server connections
- **File Transfer** - Upload and download files with progress tracking
- **Cross-Platform** - Works on macOS, Windows, and Linux
- **Cloud Sync** - Sync your host configurations across devices

## Installation

Download the latest release for your platform from the releases page.

### Build from Source

```bash
# Clone the repository
git clone https://github.com/liubaicai/SmbCatty.git
cd SmbCatty

# Install dependencies
npm install

# Start in development mode
npm run dev

# Build for production
npm run build
```
